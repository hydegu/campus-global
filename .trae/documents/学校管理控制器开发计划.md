# 学校管理控制器开发计划

## 1. DTO清单

### 1.1 SchoolAddDTO (新增学校请求DTO)
| 字段名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| orgName | String | 是 | 学校名称 |
| province | String | 是 | 省 |
| city | String | 是 | 市 |
| district | String | 是 | 区 |
| address | String | 是 | 详细地址 |
| contactPerson | String | 是 | 联系人 |
| contactPhone | String | 是 | 联系电话 |
| partnerId | Long | 否 | 合伙人ID |
| status | Integer | 是 | 状态:1启用 0禁用 |

### 1.2 SchoolQueryDTO (学校列表查询DTO)
| 字段名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| page | Integer | 是 | 页码 |
| size | Integer | 是 | 每页条数 |
| orgName | String | 否 | 学校名称 |
| province | String | 否 | 省 |
| city | String | 否 | 市 |
| district | String | 否 | 区 |
| partnerId | Long | 否 | 合伙人ID |
| partnerName | String | 否 | 合伙人名称 |
| status | Integer | 否 | 状态:1启用 0禁用 |

### 1.3 SchoolUpdateDTO (学校信息更新DTO)
| 字段名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| orgName | String | 否 | 学校名称 |
| province | String | 否 | 省 |
| city | String | 否 | 市 |
| district | String | 否 | 区 |
| address | String | 否 | 详细地址 |
| contactPerson | String | 否 | 联系人 |
| contactPhone | String | 否 | 联系电话 |
| partnerId | Long | 否 | 合伙人ID |
| status | Integer | 否 | 状态:1启用 0禁用 |

## 3. 开发步骤
1. 创建上述DTO类
2. 创建上述VO类
3. 创建SchoolService接口和实现
4. 创建SchoolController控制器
5. 测试接口功能

## 4. 技术规范
- 遵循现有项目架构和代码风格
- 使用Spring Boot和MyBatis Plus
- 使用@Operation和@Tag注解生成API文档
- 使用@HasPermission注解进行权限控制
- 使用Result类封装返回结果
- 实现学校列表查询时，支持根据学校名称和合伙人名称进行搜索
